# GoBeansproxy

A proxy for Gobeansdb.

# Install

```
$ cd ${GOHOME}
$ git clone http://github.intra.douban.com/coresys/gobeansproxy.git src/github.intra.douban.com/coresys/gobeansproxy
$ cd src/github.intra.douban.com/coresys/gobeansproxy
$ make
```

# test

```
$ make test  # unit test
```

# run

```
$ ${GOHOME}/bin/proxy -h
```
